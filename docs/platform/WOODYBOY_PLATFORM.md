# WOODYBOY Platform

ELKASS pozostaje pierwszym wdrożeniem referencyjnym. Docelowo WOODYBOY ma być nadrzędną platformą, z której będą powstawały kolejne sklepy.

## Zasada

- ELKASS = projekt / wdrożenie / dane sklepu.
- WOODYBOY Core = moduły wspólne: Product Engine, Media Studio, Theme Engine, Layout Builder, Orders, Customers, SEO, Search.

## Zasada dla kolejnych sprintów

Nowe funkcje piszemy tak, aby możliwe było ich późniejsze wydzielenie do WOODYBOY Core, ale nie zatrzymujemy prac nad wersją produkcyjną ELKASS.
