# Jak wgrać czystą strukturę

1. Zrób commit obecnego repo jako backup.
2. Pobierz tę paczkę i rozpakuj.
3. W folderze lokalnego repo usuń stare luźne pliki z root.
4. Wklej zawartość tej paczki do folderu repo.
5. GitHub Desktop pokaże usunięcia i dodania plików.
6. Commit: `v4.0 repo clean + checkout`.
7. Push origin.

Nie wklejaj tej paczki na stare pliki bez usuwania, bo stare audyty i luźne HTML zostaną w repo.
