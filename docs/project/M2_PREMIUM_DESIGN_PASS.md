# M2 – ELKASS Premium Design Pass

## Cel
Podniesienie jakości wizualnej sklepu bez dokładania nowych funkcji. Ten etap skupia się na pierwszym wrażeniu, rytmie strony, typografii, proporcjach, mobile i spójności UI.

## Zakres zmian
- Nowa warstwa CSS `assets/css/m2-premium-design-pass.css`.
- Kampanijny Hero z mocniejszą warstwą lifestyle i spokojniejszym tekstem.
- Uspokojona kolorystyka: biel, czerń, czerwony wyłącznie jako główny akcent CTA.
- Lepsze proporcje sekcji Home.
- Premium hover dla kart produktów, promocji i kategorii.
- Lżejsza sekcja producentów.
- Opinie w stylu lokalnego zaufania, bez ciężkich Google boxów.
- Produkt: czystsza cena, finansowanie, parametry i usługi.
- Mobile: niższy Hero, prostsze CTA, kategorie 2×2, opinie jako jedna karta, lepszy sticky buy bar.

## Zasada projektowa
ELKASS ma wyglądać jak lokalny sklep premium, nie jak kolejna generyczna makieta e-commerce.

## WOODYBOY
Zmiany są dopisane jako warstwa design systemu możliwa do późniejszego wydzielenia do WOODYBOY Core.
