# Home Premium Final Lock

Finalny etap wizualizacji strony głównej ELKASS.

Za wzorzec przyjęto kompozycję z wizualizacji: szeroki, niski Hero, osobny pasek statystyk, poziomy Hit tygodnia, lekkie kategorie, kompaktowy pasek marek, niskie promocje, sekcja salonu, opinie Google w spokojnym układzie i spójne „Znajdź nas”.

## Zasady

- Home ma wyglądać jak gotowy sklep, nie szkielet.
- Desktop i mobile są projektowane równolegle.
- Nie dodajemy kolejnych sekcji do Home przed zamknięciem Live CMS i bazy produktów.
- Grafiki i produkty docelowo mają pochodzić z panelu / Media Studio.
