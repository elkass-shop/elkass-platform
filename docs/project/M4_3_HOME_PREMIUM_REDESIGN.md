# M4.3 Home Premium Redesign

Zakres stabilizacji po M4:

- Hero zmienione na panoramiczne, niższe i szersze.
- Usunięto wrażenie wtopionych napisów/statystyk z tła Hero przez nowy czysty obraz `assets/hero-clean-panorama.png`.
- Statystyki są osobnym paskiem pod Hero.
- Hit tygodnia jest czytelniejszy i niższy.
- Promocje wybrane przez ELKASS są szersze i niższe.
- Showroom nie dokleja automatycznie dużych sekcji do strony głównej.
- Opinie Google są lżejsze, bez ciężkiego czarnego paska.
- Znajdź nas i mapa zostały spięte w jedną, spokojniejszą sekcję.

Uwaga architektoniczna: layout pozostaje zgodny z przyszłym kierunkiem WOODYBOY Core — elementy Home powinny być docelowo sterowane przez Home/Visual Builder.
