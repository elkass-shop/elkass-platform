# ELKASS Media Studio PRO v46

Moduł porządkuje grafiki i przygotowuje bibliotekę mediów dla produktów, kategorii, banerów, galerii, marek i motywów sezonowych.

## Założenia
- jedno źródło mediów dla całej platformy,
- osobne katalogi dla produktów, kategorii, marek, galerii, motywów i dokumentów,
- media przypisywane do produktów po ID,
- przygotowanie pod WebP/AVIF i automatyczne kadrowanie,
- osobne ustawienia desktop/mobile,
- motywy sezonowe nie mogą zasłaniać treści na telefonie.
