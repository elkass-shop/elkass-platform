# V56 — Media + Home Integration

Cel: wszystkie grafiki strony głównej mają pochodzić z Media Studio, a nie być wpisywane na sztywno w kodzie.

Dodano:
- bibliotekę `data/media-home-v56.json`,
- zakładkę `Media ↔ Home` w `/admin`,
- strukturę użycia grafik: Hero, kategorie, banery, motywy, galeria,
- przygotowanie pod Supabase Storage.

Zasada: kod trzyma strukturę i logikę, a Media Studio trzyma wybór grafik dla projektu ELKASS.
